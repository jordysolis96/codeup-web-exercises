What is the difference between display: none and display: hidden?

display: hidden

  • syntax- visibility: hidden;
  • Hides an element
  • Still affects the layout of the screen
  • Still takes up the same space on the screen as before

display: none

  • syntax- display: none
  • Hides an element
  • Element no longer takes up any space on the screen
